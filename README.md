# Portafolio web – Mariana Hernández

Portafolio de una desarrolladora web junior donde se presentan sus habilidades, herramientas principales y una selección de proyectos destacados. El diseño está inspirado en una tarjeta oscura con degradados y enfoque en la legibilidad del contenido.

La página está maquetada con HTML semántico y estilos responsivos en CSS, cuidando la accesibilidad y la apariencia en diferentes tamaños de pantalla (móvil, tableta y escritorio).

## Tecnologías y técnicas utilizadas

- HTML5 semántico (header, main, section, article, footer, figure, nav)
- CSS3 moderno
  - Flexbox y Grid para la composición del layout
  - Consultas de medios para los puntos de ruptura 320px, 768px y 1440px
  - Degradados y sombras para recrear el diseño oscurecido
- Normalize CSS para un estilo base consistente entre navegadores
- Tipografías: Archivo Black y Open Sans, con fuentes del sistema como alternativas
